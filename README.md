# Pr-diction-Conforme-et-R-gression-Quantile-sur-le-Risque-de-Cr-dit---Finance
Ce projet s’inscrit dans le cadre d’une étude sur la **quantification de l’incertitude** dans les modèles de Machine Learning.   Ces méthodes sont appliquées au jeu de données **Statlog (German Credit Data)** afin d’évaluer leur performance et leur robustesse dans un contexte de **gestion du risque bancaire**.
